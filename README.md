
# マルコフ連鎖による文章の自動生成

### 要件

- Python3
- MeCab ... `pip install mecab-python3`

### 構成

- input.txt : マルコフ連鎖を作るもとになる文書データ
- main.py : 文章生成プログラム

入力の文書データにはWikipediaの暗号技術関連の記事から適当に拾ってきました。

### 実行方法

```
$ python main.py
```
